## 📝 Problem Statement

You are given a string `word`. A letter is called **special** if it appears both in lowercase and uppercase in `word`.

Return the *number of special letters* in `word`.

<br/>

## 🎯 Constraints

- `1 <= word.length <= 50`
- `word` consists of only lowercase and uppercase English letters.

<br/>

## 💡 Examples

| Example | Input | Output | Explanation |
| :--- | :--- | :--- | :--- |
| **1** | `word = "aaAbcBC"` | `3` | The special characters in `word` are `'a'`, `'b'`, and `'c'`. |
| **2** | `word = "abc"` | `0` | No character in `word` appears in uppercase. |
| **3** | `word = "abBCab"` | `1` | The only special character in `word` is `'b'`. |

<br/>

## 🧠 Approach / Explanation

A letter is considered "special" if its lowercase version and uppercase version both exist within the given string. Since strings only contain English letters, we can use a fixed-size frequency array to keep track of the letters we've encountered. 

We can solve this problem in multiple ways:

1. **Two-Pass Approach (Using ASCII values):** We iterate through the string once to mark the occurrences of every character in a frequency array of size 123 (to cover ASCII up to 'z'). Then, we iterate from the ASCII values `97` to `122` (i.e., `'a'` to `'z'`) and check if both the lowercase character `freq[i]` and its uppercase equivalent `freq[i - 32]` exist.
2. **Two-Pass Approach (Using Characters):** Similar to the previous approach, but instead of raw integers, we loop using character iteration (`'a'` to `'z'`) making the code slightly more readable.
3. **One-Pass Approach:** As we iterate through the string, we increment the frequency of the current character. If it is the exact *first time* we are seeing this character, we immediately check if its opposite case (uppercase for lowercase, or lowercase for uppercase) has already been encountered. If it has, we increment our special character count.

## 💻 Solution Code

### 🟢 Approach 1: Frequency Count using ASCII index (Two Pass)

```cpp
// Intuition: We can count the occurrences of each character using a frequency array. Checking for special characters then involves verifying if both lowercase and uppercase ASCII values have non-zero counts.
// Approach: Create a frequency array of size 123. Iterate over the string to record each character's presence. Then, loop through lowercase ASCII values (97 to 122) and check if both the lowercase and its uppercase counterpart (i - 32) exist.
// Time Complexity: O(n), where n is the length of the string, as we make one pass to populate the array and a constant 26-step loop.
// Space Complexity: O(1), since we use a fixed-size frequency array of 123 elements.
class Solution {
public:
    int numberOfSpecialChars(string word) {
        int freq[123] = {0}; // frequency array initialized to 0

        for(char ch : word) {
            freq[ch]++; // record presence
        }

        int count = 0;
        // ASCII : a : 97 , b = 98 ..... 122 : z
        // ASCII : A : 65,  B = 66 ..... 90  : Z

        for(int i = 97; i <= 122; i++) { // Iterate from 97 to 122 or 'a' to 'z'
            int capitalIndex = i - 32; // A... Z

            if(freq[i] > 0 && freq[capitalIndex] > 0) // Both should be present
                count++;
        }

        return count;
    }
};

/*
*
* Dry Run
* word = "aaAbcBC"
* Pass 1: freq array populates: freq['a']=2, freq['A']=1, freq['b']=1, freq['c']=1, freq['B']=1, freq['C']=1
* Pass 2:
* i=97 ('a'): freq['a']>0 & freq['A']>0 -> count = 1
* i=98 ('b'): freq['b']>0 & freq['B']>0 -> count = 2
* i=99 ('c'): freq['c']>0 & freq['C']>0 -> count = 3
* Other letters have freq 0. Return count = 3.
*
*/
```

### 🟡 Approach 2: Frequency Count using Char Iteration (Two Pass)

```cpp
// Intuition: This is logically identical to Approach 1, but uses character literals instead of raw ASCII numbers for better readability and cleaner syntax.
// Approach: Populate the frequency array in one pass. Then iterate a character variable from 'a' to 'z', compute its uppercase equivalent, and check if both exist in the frequency array.
// Time Complexity: O(n), mapping the string length taking linear time, plus a constant 26 loop.
// Space Complexity: O(1), fixed-size array space.
class Solution {
public:
    int numberOfSpecialChars(string word) {
        int freq[123] = {0}; // fixed size frequency tracker

        for(char ch : word) {
            freq[ch]++; // count occurrences
        }

        int count = 0;
        // ASCII : a : 97 , b = 98 ..... 122 : z
        // ASCII : A : 65,  B = 66 ..... 90  : Z

        for(char ch = 'a'; ch <= 'z'; ch++) { // Iterate from 'a' to 'z'
            char capital = ch - 'a' + 'A'; // Determine uppercase counterpart

            if(freq[ch] > 0 && freq[capital] > 0) // Both should be present
                count++;
        }

        return count;
    }
};

/*
*
* Dry Run
* word = "abc"
* Pass 1: freq['a']=1, freq['b']=1, freq['c']=1
* Pass 2:
* ch='a': freq['a']>0 but freq['A']==0 -> no match
* ch='b': freq['b']>0 but freq['B']==0 -> no match
* ch='c': freq['c']>0 but freq['C']==0 -> no match
* Return count = 0.
*
*/
```

### 🔴 Approach 3: Optimized One Pass Frequency Count

```cpp
// Intuition: Instead of looping twice, we can dynamically check for pairs while traversing the string. If a character appears for the very first time, we check if its opposite case is already recorded.
// Approach: Iterate the string and increment the character's frequency. If it's the first time seeing this exact character (freq == 1), calculate its alternate case. If the alternate case's frequency is > 0, we found a pair and increment count.
// Time Complexity: O(n), as we process the string in exactly one single pass.
// Space Complexity: O(1), fixed-size frequency array.
class Solution {
public:
    int numberOfSpecialChars(string word) {
        int freq[123] = {0}; // keeps track of encountered characters
        int count = 0;

        for(char ch : word) {
            freq[ch]++; // mark character as seen

            // Current ch is lower case, then look for upper case
            if(ch >= 'a' && ch <= 'z') {
                char capital = ch - 'a' + 'A';
                if(freq[ch] == 1 && freq[capital] > 0) // lowercase just appeared for first time
                    count++;
            } else { // Current ch is upper case, then look for lower case
                char lower = ch - 'A' + 'a';
                if(freq[ch] == 1 && freq[lower] > 0) // uppercase just appeared for first time
                    count++;
            }
        }

        return count;
    }
};

/*
*
* Dry Run
* word = "abBCab"
* ch='a': freq['a']=1, freq['A']=0. count=0
* ch='b': freq['b']=1, freq['B']=0. count=0
* ch='B': freq['B']=1. Is freq['b']>0? Yes! count=1
* ch='C': freq['C']=1, freq['c']=0. count=1
* ch='a': freq['a']=2. First time condition fails. count=1
* ch='b': freq['b']=2. First time condition fails. count=1
* Return count = 1.
*
*/
```

## 📌 Notes & Key Insights

- **ASCII Arithmetic:** Using `ch - 'a' + 'A'` or `i - 32` is a fast and efficient way to map a lowercase letter to its uppercase counterpart in languages like C/C++.
- **Frequency Arrays:** For problems involving English letters, allocating a small fixed-size array (like `123` or `256`) is faster and uses less memory overhead than `std::unordered_map`.
- **First Encounter Tracking:** In the one-pass approach, checking `freq[ch] == 1` guarantees that we don't accidentally count the same special character multiple times if the string contains duplicate sequences (e.g., `"aAaA"`).
