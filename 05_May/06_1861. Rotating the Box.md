# 1861. Rotating the Box

**Difficulty:** Medium  
**Topics:** `Array`, `Two Pointers`, `Matrix`, `Simulation`

---

## 📝 Problem Statement

You are given an `m x n` matrix of characters `boxGrid` representing a side-view of a box. Each cell of the box is one of the following:

* A stone `"#"`
* A stationary obstacle `"*"`
* Empty `"."`

The box is rotated **90 degrees clockwise**, causing some of the stones to fall due to gravity. Each stone falls down until it lands on an obstacle, another stone, or the bottom of the box. Gravity **does not** affect the obstacles' positions, and the inertia from the box's rotation **does not** affect the stones' horizontal positions.

It is **guaranteed** that each stone in `boxGrid` rests on an obstacle, another stone, or the bottom of the box.

Return an `n x m` matrix representing the box after the rotation described above.

## 🚀 Constraints

* `m == boxGrid.length`
* `n == boxGrid[i].length`
* `1 <= m, n <= 500`
* `boxGrid[i][j]` is either `'#'`, `'*'`, or `'.'`.

---

## 🧠 Key Insights

* **Decouple the Problem:** The problem can be logically split into two independent tasks: rotating the matrix by 90 degrees clockwise, and then applying gravity to the stones.
* **Matrix Rotation:** A 90-degree clockwise rotation of any matrix can be efficiently achieved by first finding its transpose (swapping rows and columns) and then reversing every row.
* **Optimized Gravity (Two Pointers):** Instead of searching upwards for a stone every time an empty space is found (which takes $O(N^2)$ per column), we can iterate from the bottom up. By keeping a pointer to the "lowest available empty space", we can shift stones down in a single $O(N)$ pass per column.

---

## 💡 Approach

### 1. Brute Force Simulation
First, rotate the box 90 degrees. Create an `n x m` matrix. To get the 90-degree rotated view, transpose the original matrix and reverse each row. Once rotated, apply gravity. For each column, iterate from the bottom row to the top. If you find an empty space `'.'`, iterate further upwards to find the nearest stone `'#'`. If you encounter an obstacle `'*'` before a stone, stop looking for that space. Swap the stone with the empty space. 

### 2. Two-Pointer Optimization
The rotation step remains the same. To optimize the gravity simulation, process each column independently from bottom to top. Maintain a pointer `spaceBottomRow` that tracks the lowest available empty slot in the current column. As you iterate upwards:
* If you see an obstacle `'*'`: Stones cannot fall through it. Update `spaceBottomRow` to just above the obstacle (`i - 1`).
* If you see a stone `'#'`: Move this stone to `spaceBottomRow`, empty the current cell (set to `'.'`), and decrement `spaceBottomRow` by 1 so the next falling stone lands right above it.

```cpp
// Intuition: Divide the problem into two parts: rotate the matrix 90 degrees, then simulate gravity. For gravity, when an empty space is found, search upwards for a stone to drop into it.
// Approach:
// 1. Create a result matrix of size n x m.
// 2. Transpose the original box into the result matrix.
// 3. Reverse each row of the result matrix to complete the 90-degree clockwise rotation.
// 4. Iterate through each column from bottom to top. If an empty space ('.') is found, run a nested loop upwards to find the closest stone ('#') before hitting an obstacle ('*'). Swap them.
// Time Complexity: O(m * n^2) - Rotating takes O(m * n). Applying gravity takes O(cols * rows * rows) due to the nested upward search.
// Space Complexity: O(1) auxiliary space, excluding the O(m * n) space required for the result matrix.
class Solution {
public:
    vector<vector<char>> rotateTheBox(vector<vector<char>>& box) {
        int m = box.size();
        int n = box[0].size();

        vector<vector<char>> result(n, vector<char>(m));

        // Transpose the matrix
        for(int i = 0; i < n; i++) {
            for(int j = 0; j < m; j++) {
                result[i][j] = box[j][i];
            }
        }

        // Reverse each row for 90-degree clockwise rotation
        for(vector<char>& row : result) {
            reverse(begin(row), end(row));
        }

        // Apply gravity
        for(int j = 0; j < m; j++) {
            for(int i = n - 1; i >= 0; i--) {
                if(result[i][j] == '.') { 
                    int stoneRow = -1;

                    // Search upwards for a stone
                    for(int k = i - 1; k >= 0; k--) {
                        if(result[k][j] == '*') {
                            break; // Obstacle blocks the path
                        } else if(result[k][j] == '#') {
                            stoneRow = k;
                            break; // Stone found
                        }
                    }

                    // Drop the stone
                    if(stoneRow != -1) {
                        result[i][j] = '#';
                        result[stoneRow][j] = '.';
                    }
                }
            }
        }

        return result;
    }
};

*
* Dry Run
*
* Input: boxGrid = [["#",".","#"]] (1x3)
* Transpose & Reverse -> Result becomes 3x1: 
* [["#"], 
*  ["."], 
*  ["#"]]
*
* Gravity processing (col 0):
* i = 2: result[2][0] == '#' (Skip)
* i = 1: result[1][0] == '.' (Empty space found)
*        Search upwards: k = 0, result[0][0] == '#' (Stone found!)
*        stoneRow = 0. Swap result[1][0] and result[0][0].
* Result after gravity:
* [["."], 
*  ["#"], 
*  ["#"]]
* 
* Return result.
*/
```

```cpp
// Intuition: Rotating the box is standard. We can optimize gravity by maintaining a pointer to the lowest empty space in each column, avoiding the nested upward search.
// Approach:
// 1. Transpose and reverse the rows to achieve a 90-degree clockwise rotation.
// 2. Iterate through each column. Maintain `spaceBottomRow` pointing to the lowest available row.
// 3. Traverse from bottom to top. If an obstacle is found, update `spaceBottomRow` to just above it. If a stone is found, drop it to `spaceBottomRow` and decrement the pointer.
// Time Complexity: O(m * n) - Matrix rotation takes O(m * n). Gravity is applied in a single pass per column, taking O(m * n).
// Space Complexity: O(1) auxiliary space, excluding the O(m * n) space required for the result matrix.
class Solution {
public:
    vector<vector<char>> rotateTheBox(vector<vector<char>>& box) {
        int m = box.size();
        int n = box[0].size();

        vector<vector<char>> result(n, vector<char>(m));

        // Transpose the matrix
        for(int i = 0; i < n; i++) {
            for(int j = 0; j < m; j++) {
                result[i][j] = box[j][i];
            }
        }

        // Reverse each row for 90-degree clockwise rotation
        for(vector<char>& row : result) {
            reverse(begin(row), end(row));
        }

        // Apply gravity using a two-pointer approach
        for(int j = 0; j < m; j++) {
            int spaceBottomRow = n - 1; // Tracks the lowest available empty slot
            
            for(int i = n - 1; i >= 0; i--) {
                if(result[i][j] == '*') {
                    // Obstacle resets the lowest available slot to just above it
                    spaceBottomRow = i - 1;
                    continue;
                }

                if(result[i][j] == '#') {
                    // Drop the stone to the lowest available slot
                    result[i][j] = '.';
                    result[spaceBottomRow][j] = '#';
                    spaceBottomRow--; // Shift the available slot up by one
                }
            }
        }

        return result;
    }
};

/*
*
* Dry Run
*
* Input: box = [["#",".","*","."], ["#","#","*","."]]
* Dimensions: m=2, n=4. Result will be 4x2.
*
* After Transpose & Reverse (90-degree rotation):
* [["#","."],
*  ["#","#"],
*  ["*","*"],
*  [".","."]]
* 
* Apply Gravity:
* Col 0: spaceBottomRow = 3
*   i=3 ('.'): continue
*   i=2 ('*'): spaceBottomRow = 1
*   i=1 ('#'): swap with spaceBottomRow(1) -> no change, spaceBottomRow = 0
*   i=0 ('#'): swap with spaceBottomRow(0) -> no change, spaceBottomRow = -1
* Col 1: spaceBottomRow = 3
*   i=3 ('.'): continue
*   i=2 ('*'): spaceBottomRow = 1
*   i=1 ('#'): swap with spaceBottomRow(1) -> no change, spaceBottomRow = 0
*   i=0 ('.'): continue
* 
* Final Result:
* [["#","."],
*  ["#","#"],
*  ["*","*"],
*  [".","."]]
*/
```
