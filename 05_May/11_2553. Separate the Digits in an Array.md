# 2553. Separate the Digits in an Array

## 📝 Problem Statement

Given an array of positive integers `nums`, return an array `answer` that consists of the digits of each integer in `nums` after separating them in **the same order** they appear in `nums`.

To separate the digits of an integer is to get all the digits it has in the same order.
* For example, for the integer `10921`, the separation of its digits is `[1,0,9,2,1]`.

---

## 🚦 Constraints

* `1 <= nums.length <= 1000`
* `1 <= nums[i] <= 10^5`

---

## 💡 Examples

| Example | Input | Output | Explanation |
| :--- | :--- | :--- | :--- |
| **1** | `nums = [13,25,83,77]` | `[1,3,2,5,8,3,7,7]` | The separation of `13` is `[1,3]`.<br>The separation of `25` is `[2,5]`.<br>The separation of `83` is `[8,3]`.<br>The separation of `77` is `[7,7]`.<br>Resulting in `[1,3,2,5,8,3,7,7]`. |

## 🧠 Approach

<details>
<summary><b>Click to expand the step-by-step walkthrough</b></summary>

1. **Initialization:** Create an empty dynamic array (vector) named `result` to store the final sequence of digits.
2. **Iteration:** Loop through each number `num` in the given `nums` array.
3. **Digit Extraction:** For each `num`, extract its digits from right to left using the modulo operator (`num % 10`). Push these digits into a temporary `digits` array and divide `num` by `10` until it becomes `0`.
4. **Reversal & Storage:** Because the digits were extracted in reverse order (least significant digit first), loop through the temporary `digits` array backwards, pushing each digit into the final `result` array.
5. **Return:** After processing all numbers in `nums`, return the `result` array.

</details>

## 💻 Solution Code

```cpp
// Intuition: We need to extract the individual digits of each number and place them in our result array in their original order. Extracting digits using modulo gives them to us in reverse, so we can store them temporarily and append them backwards.
// Approach: Iterate through each number in the array. Use a `while` loop with modulo 10 to extract digits from right to left into a temporary vector. Then, iterate backwards through this temporary vector to append the digits into the final result vector in the correct left-to-right order.
// Time Complexity: O(N * log M), where N is the number of elements in `nums` and log M is the number of digits in the maximum element. We process each digit of every number exactly once.
// Space Complexity: O(log M) for the temporary `digits` vector used to store the digits of a single number before reversing them. The space for the `result` array is O(N * log M) but is not counted towards auxiliary space.

class Solution {
public:
    vector<int> separateDigits(vector<int>& nums) {
        vector<int> result; // To store the final separated digits

        for (auto& num : nums) {
            vector<int> digits; // Temporary storage for digits of the current number
            
            // Extract digits right-to-left
            while (num > 0) {
                digits.push_back(num % 10); // Get the last digit
                num /= 10; // Remove the last digit from the number
            }

            // Append digits to result in original left-to-right order
            for (int i = digits.size() - 1; i >= 0; i--) {
                result.push_back(digits[i]);
            }
        }
        return result;
    }
};

/*
*
* Dry Run
* Input: nums = [13, 25]
* 
* Iteration 1: num = 13
*   - num > 0 (13 > 0): push 13 % 10 (3) to digits. digits = [3]. num becomes 1.
*   - num > 0 (1 > 0): push 1 % 10 (1) to digits. digits = [3, 1]. num becomes 0.
*   - Reverse loop over digits: push digits[1] (1), then digits[0] (3) to result.
*   - result = [1, 3]
* 
* Iteration 2: num = 25
*   - num > 0 (25 > 0): push 25 % 10 (5) to digits. digits = [5]. num becomes 2.
*   - num > 0 (2 > 0): push 2 % 10 (2) to digits. digits = [5, 2]. num becomes 0.
*   - Reverse loop over digits: push digits[1] (2), then digits[0] (5) to result.
*   - result = [1, 3, 2, 5]
* 
* Final Output: [1, 3, 2, 5]
*
*/
```

## 🔍 Key Insights
* Using `num % 10` paired with `num /= 10` is the most mathematically efficient way to isolate digits from an integer without converting it to a string.
* Storing the extracted digits in a temporary stack or array allows for easy reversal to maintain the exact visual order from the original array.
