## 📖 Problem Statement

A string `originalText` is encoded using a **slanted transposition cipher** to a string `encodedText` with the help of a matrix having a **fixed number of rows** `rows`.

`originalText` is placed first in a top-left to bottom-right manner.

The blue cells are filled first, followed by the red cells, then the yellow cells, and so on, until we reach the end of `originalText`. The arrow indicates the order in which the cells are filled. All empty cells are filled with `' '`. The number of columns is chosen such that the rightmost column will **not be empty** after filling in `originalText`.

`encodedText` is then formed by appending all characters of the matrix in a row-wise fashion.

Given the encoded string `encodedText` and number of rows `rows`, return the original string `originalText`.

> **Note:** `originalText` does **not** have any trailing spaces `' '`. The test cases are generated such that there is only one possible `originalText`.

---

## 🎯 Constraints

- `0 <= encodedText.length <= 10^6`
- `encodedText` consists of lowercase English letters and `' '` only.
- `encodedText` is a valid encoding of some `originalText` that **does not** have trailing spaces.
- `1 <= rows <= 1000`
- The testcases are generated such that there is **only one** possible `originalText`.

## 🧠 Explanation & Approach

The problem asks us to reverse a slanted transposition cipher. The encoded text was generated by taking a string, placing its characters diagonally in a grid, and then reading the grid row by row. 

To decode it, we must realize a key mathematical relationship:
1. **Find Grid Dimensions:** We are given the total number of characters (`encodedText.length()`) and the number of `rows`. Since the grid is fully filled (with spaces where necessary), the number of columns will always be `encodedText.length() / rows`.
2. **Recreate the Diagonals:** - **Brute Force (Simulation):** We can literally recreate the 2D matrix of size `rows x columns`, fill it row by row using `encodedText`, and then read it diagonally starting from the top row (`i = 0`, `j = col`).
   - **Optimized (Index Jumping):** We don't actually need the 2D matrix. In the 1D string `encodedText`, moving "down one row and right one column" corresponds to jumping exactly `columns + 1` characters ahead. Therefore, we can start at each column index `col` in the first row, and repeatedly add `columns + 1` to our index to pick the diagonal elements directly.
3. **Remove Trailing Spaces:** Because the original text was padded with spaces to fit the grid, we must strip all trailing spaces from our final decoded string.

## 💻 Brute-force Solution (C++)

```cpp
// Intuition: If we simply recreate the 2D matrix that was used to encode the text, we can trace the original placement of characters by reading the matrix diagonally.
// Approach: 1. Calculate columns = length / rows. 2. Fill a 2D matrix row by row. 3. Traverse diagonally starting from each column of the first row. 4. Remove trailing spaces from the final string.
// Time Complexity: O(l), where l is the length of encodedText. We visit each character to build the matrix and then read it.
// Space Complexity: O(l), to store all characters in the 2D matrix simulation.

class Solution {
public:
    string decodeCiphertext(string encodedText, int rows) {
        // Base case: if there's only 1 row, no transposition happens
        if (rows == 1) 
            return encodedText;

        int l       = encodedText.size();
        int columns = l / rows;

        string originalText = "";

        // Create matrix to hold the encoded characters
        vector<vector<char>> mat(rows, vector<char>(columns));

        int idx = 0;
        // Fill the matrix row by row as it was originally read
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < columns; j++) {
                mat[i][j] = encodedText[idx++];
            }
        }

        // Read diagonally to reconstruct original text
        for (int col = 0; col < columns; col++) {
            int i = 0;
            int j = col;

            // Traverse the current diagonal
            while (i < rows && j < columns) {
                originalText += mat[i][j];
                i++;
                j++;
            }
        }

        // Remove trailing spaces added during encoding padding
        while (!originalText.empty() && originalText.back() == ' ') {
            originalText.pop_back();
        }

        return originalText;
    }
};

/*
*
* Dry Run
* encodedText = "ch   ie   pr", rows = 3
* l = 12, cols = 12/3 = 4
* Matrix filled:
* [c] [h] [ ] [ ]
* [ ] [i] [e] [ ]
* [ ] [p] [r] [ ]
* Diagonals:
* col 0: (0,0)->c, (1,1)->i, (2,2)->r => "cir" (Wait, matrix filling is sequential)
* Let's trace proper index:
* mat fills as:
* c h _ _
* _ i e _
* _ p r _
* Diagonal col=0: (0,0)=c, (1,1)=i, (2,2)=p => "cip"
* Diagonal col=1: (0,1)=h, (1,2)=e, (2,3)=_ => "he "
* Diagonal col=2: (0,2)=_, (1,3)=_, out => "  "
* Diagonal col=3: (0,3)=_, out => " "
* originalText = "cipher   "
* Trailing space removal => "cipher"
*
*/
```

## 🚀 Optimized Solution (C++)

```cpp
// Intuition: Instead of building the matrix, we can simulate the diagonal traversal directly on the 1D encoded string. Moving diagonally (down 1 row, right 1 col) maps to a jump of `columns + 1` in the 1D string.
// Approach: 1. Calculate columns. 2. Iterate starting from each column `col` from 0 to `columns-1`. 3. Jump indices by `columns + 1` to collect characters diagonally. 4. Strip trailing spaces.
// Time Complexity: O(l), where l is the length of encodedText. We visit each character exactly once.
// Space Complexity: O(1), no extra matrix is created, only the result string is returned.

class Solution {
public:
    string decodeCiphertext(string encodedText, int rows) {
        int l = encodedText.length();

        // Total columns in the implicit grid
        int columns = l / rows;

        string originalText;
        
        // Start from each column in the first row
        for(int col = 0; col < columns; col++) {
            // Jump by (columns + 1) to simulate diagonal movement
            for(int j = col; j < l; j += (columns + 1)) {
                originalText += encodedText[j];
            }
        }

        // Remove trailing padding spaces
        while(!originalText.empty() && originalText.back() == ' ') {
            originalText.pop_back();
        }

        return originalText;
    }
};

/*
*
* Dry Run
* encodedText = "ch   ie   pr", rows = 3
* l = 12, cols = 12/3 = 4
* Jump size = cols + 1 = 5
* col 0: j=0 ('c'), j=5 ('i'), j=10 ('p') -> "cip"
* col 1: j=1 ('h'), j=6 ('e'), j=11 ('r') -> "her"
* col 2: j=2 (' '), j=7 (' '), j=12 (out) -> "  "
* col 3: j=3 (' '), j=8 (' '), j=13 (out) -> "  "
* Result before pop = "cipher    "
* Result after popping spaces = "cipher"
*
*/
```
